# WordleCracker
## Description:
* Specialized for 5 words wordle, terminal based
* Enter 0-4 for operations
    * 0: leave
    * 1:correct position
    * 2:incorrect position
    * 3:character dne
    * 4:handle 5 character (an entry) at same time
## Example:
    input:
    4
    ninja
    10233
    
    output:
    Words that matches the existing hints
